## 📞 Support Resources

| Issue | Solution |
|-------|----------|
| Port 3000 busy | `npm run dev -- -p 3001` |
| Dependencies error | `npm cache clean --force && npm install` |
| TypeScript errors | `npm run type-check` |
| Build fails | `rm -rf .next && npm run build` |

See `DEPLOYMENT.md` for more troubleshooting.
